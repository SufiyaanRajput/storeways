## Storeways Documentation

This package hosts the Storeways documentation site, built with Docusaurus.

### Commands

```bash
pnpm docs:dev    # start local docs server
pnpm docs:build  # build static site
pnpm docs:serve  # serve the build locally
```

Docs are available at the `/docs` route (e.g., http://localhost:3002/docs when running locally in this package).
