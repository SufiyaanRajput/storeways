---

The Client app is a Next.js storefront for Storeways.

## Run Locally

```bash
pnpm dev:client
```

Open:

```
http://localhost:3000
```

## Build

```bash
pnpm --filter client run build
```

## Notes

- Bootstrapped with Next.js
- Theming and templates under `repos/client/themes`
