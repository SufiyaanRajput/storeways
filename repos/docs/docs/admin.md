The Admin app lets you manage products, categories, orders, and users.

## Run Locally

From the repo root:

```bash
pnpm dev:admin
```

The admin will be available at:

```
http://localhost:3001
```

## Build

```bash
pnpm --filter admin run build
```

## Notes

- Built with Create React App (React 17)
- Uses Ant Design and styled-components
