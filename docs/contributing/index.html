Contributing

We welcome contributions! You can:

  • Report bugs and request features via GitHub Issues
  • Submit pull requests for fixes and features
  • Add plugins (storage, payments, analytics, etc.)

Quick Start for Contributors​

corepack enable
pnpm setup-p
pnpm dev

Development Tips​

  • Keep changes focused and well‑scoped
  • Add tests or examples when possible
  • Update docs if behavior changes

Pull Requests​

  1. Fork and create a feature branch
  2. Make changes with clear commits
  3. Ensure apps build and run
  4. Open a PR with a concise description and screenshots if applicable

Code of Conduct​

Please be respectful and professional in all interactions. We follow common OSS standards.
